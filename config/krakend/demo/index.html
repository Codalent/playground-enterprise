Documentation

OpenAPI/Swagger documentation can be generated with krakend generate openapi

docker-compose exec krakend_ee krakend generate openapi -o /etc/krakend/demo/swagger.json
Postman

OpenAPI/Swagger documentation can be generated with krakend generate openapi

docker-compose exec krakend_ee krakend generate postman -o /etc/krakend/demo/postman.json
Identity Providers

Keycloak and many other identity providers can be integrated with KrakenD

API-Key authentication

The API key authentication enables a Role-Based Access Control (RBAC) mechanism by reading the Authorization header of incoming requests.

curl -iG -H 'Authorization: Bearer 58427514-be32-0b52-b7c6-d01fada30497' http://localhost:8080/api-key

Other Enterprise features

Basic Authentication

Protect the access to selected endpoints using basic username and password credentials. or video conferencing meetings.

Virtual Hosts

Run different configurations of KrakenD endpoints per host accessing the server.

WebSockets integration

Communications using the WebSocket Protocol (RFC-6455) to enable two-way communication between a client to a backend host through the API gateway.

Wildcards

URLs starting with known patterns are forwarded to a common endpoint, without needing to declare all possible routes.

GeoIP integration

Load Maxmind’s GeoIP2 City database (payment and free versions) and enrich all KrakenD calls to your backends with geo data.

Global rate limit

Enable a Redis database store to centralize all KrakenD node rate-limit counters.

Static content

Serve static content to your users, such as JS, CSS or images, or let KrakenD consume static content as part of its backend interaction.

Multiple Identity Providers

KrakenD validates tokens issued by multiple Identity Providers..

OpenAPI / Swagger + Inspector

Automatic generation of API documentation and inspection of backend interaction through KrakenD Studio

OpenAPI CLI

Automatic generation of API documentation via command line command. For CI/CD.

Postman collections

Automatic generation of Postman collections.

URL Rewrite

Declare a dictionary of additional URLs that map to existing endpoints.

Config to png image

A generated image of the configuration might help you understand better the running configuration.

Client redirects

Avoid following redirects of the backends and let the end-user do it instead.
